Flash2learn

An easy to use android study review app that allows users to create, edit, save, and remove items.

First Iteration

First Demo

User Stories

  • Front side for card to display the question.
  • Back side for car to display the answer.
  • Color code to show correct answer from options provided.
  • Style the card.
  • Button to toggle between question and answer side of card.

Second Iteration

Second Demo

User Stories

  • User can click on '+' button to navigate to a new screen.
  • User can enter and submit a question in the Add card screen.
  • User can use save button to save a newly created card.
  • User can edit an already created card.
  • Notification appears when a new card is added to the deck.
  • User can enter multiple choice answers in the Add card screen.

Third Iteration

Third Demo

User Stories

  • User can create card and see the card when the app is relaunched.
  • User can create multiple cards and browse through the cards.
  • User can delete a card from the deck.
  • Additional styling added to customize the user interface.

Final Iteration

Final Demo

User Stories

  • User observes a transition animation when navigating main screen to add card screen.
  • User sees a reveal animation when the flashcard is tappped and answered.
